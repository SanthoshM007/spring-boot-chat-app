Project Overview:

This project is a real-time chat application built using Spring Boot and WebSocket.

Features:

Real-time messaging
User authentication
Multiple chat rooms
Typing indicators
Message timestamps
User presence status
Technologies Used:

Java
Spring Boot
WebSocket
Thymeleaf
Bootstrap
Maven
